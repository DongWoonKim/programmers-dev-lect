## 📌 마크다운(MarkDown)

일반 텍스트 문서의 양식을 편집하기 위해 만들어진 경량 마크업 언어입니다.
특수 기호와 문자를 활용한 매우 간단한 문법으로, 마우스를 사용해 글꼴이나 크기를 바꾸는 워드프로세서와 달리 키보드 타이핑만으로 직관적으로 문서를 작성할 수 있습니다.
<details>

- [마크다운 매뉴얼](https://gist.github.com/ihoneymon/652be052a0727ad59601)
- [마크다운 테스터](https://stackedit.io/app)

# 제일 큰 사이즈 제목

- 목록 아이템 3
